Webex Calling Detailed Report App

Overview

This application automates the generation of detailed reports from Webex Calling. It provides an efficient solution for analyzing Call Detail Record (CDR) data, focusing on enhancing productivity and communication efficiency within organizations.

How it Works

1. Users authenticate with Webex Teams through a secure OAuth2 flow.

2. Upon authentication, the application fetches CDR data from Webex Calling.

3. The CDR data is processed and outputted into a .csv file.

4. Users can then use this .csv file for advanced analytics and visualization, such as with Microsoft Power BI.

This tool is designed to bridge the gap between data availability and actionable insights, aiding in data-driven decision-making.
